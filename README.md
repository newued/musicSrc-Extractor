# musicSrc-Extractor

##使用说明：

1- 找到音乐后复制网址到对应的输入框内。如果此时弹出安全性提示框，请单击确定。
2- 把提取出来的mp3真实地址复制出来用迅雷等下载工具下载出来。

使用本工具 必须在电脑上安装并支持flashPlayer。
